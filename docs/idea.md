# 权限最小粒度
  * 文章 
     ```
     发布内容
     发布指定结点内容
     发布指定站点内容
     发布指定模型内容
     编辑内容
     编辑自己内容
     编辑指定结点内容
     编辑指定模型内容
     删除内容
     删除自己内容
     删除指定结点内容
     删除指定模型内容
     删除指定站点内容
     ```
  * 内容模型管理
     ```
     新建字段
     编辑字段
     删除字段
     新建模型
     编辑模型
     删除模型
     ```
  * 站点管理
     ```
     建新站点
     编辑站点
     删除站点
     ```
  * 结点管理
     ```
     新建结点
     编辑结点
     编辑指定结点（包括子结点）
     删除结点
     删除指定结点（包括子结点）
     ```
  * 用户管理
     ```
     新增用户
     编辑用户
     编辑指定站点用户
     删除用户
     删除指定站点用户
     ```
  * 角色管理
     ```
     新建角色
     编辑角色（编辑低角色）
     删除角色（删除低角色）
     ```
